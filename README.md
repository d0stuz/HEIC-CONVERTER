HEIC Converter
HEIC Converter is a free, open-source tool that lets you quickly and easily convert .HEIC images to JPG, PNG, or WebP format on Windows.

🚀 Features
Batch Conversion: Convert hundreds of files with a single click.

Format Support: Output options include JPG, PNG, and advanced WebP (with lossless compression support).

Resizing: Resize files to any percentage or automatically scale them by specifying a maximum edge length.

Transparency Management: Preserve transparency in WebP outputs, blend with the background color, or make them opaque.

📥 How to Use?
Download the latest .exe file from the Releases page.

Run the application.

Select the source folder or files.

Specify the destination folder and select the output format.

Click the Convert button and wait for the process to complete.
